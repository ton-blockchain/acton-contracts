# Highload V3

Original FunC: [ton-blockchain/highload-wallet-contract-v3](https://github.com/ton-blockchain/highload-wallet-contract-v3).

This package implements the high-throughput wallet optimized for batched dispatch, replay protection, and bounded query-state management. Tests cover basic validation, batching and action dispatch, internal filters, replay timeout handling, storage and query cleanup limits, and end-to-end flows.
