# NFT

Original FunC: [ton-blockchain/nft-contract](https://github.com/ton-blockchain/nft-contract).

This package implements the reference NFT collection and item contracts with royalty-aware transfer and metadata flows. Tests cover collection behavior and item behavior in focused reference suites.

Scripts in `scripts/` cover deployment, state inspection, and collection or item management via `deploy.tolk`, `info.tolk`, and `manage.tolk`. Run them with `acton script nft/scripts/<name>.tolk`.
